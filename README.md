# Kasireddy Sai Kiran — Portfolio

A responsive, deployable personal portfolio built with React, TypeScript, and Vite.

## Run locally

1. Install Node.js 20 or later.
2. Run `npm install`.
3. Run `npm run dev`.
4. Open the local address shown in the terminal.

## Build for deployment

Run `npm run build`. The production-ready files are created in `dist/`.

## Deploy with Vercel

1. Create a GitHub repository and upload this project.
2. In Vercel, select **Add New → Project** and import the repository.
3. Vercel automatically recognizes Vite. Keep the default build settings and deploy.

## Updating your content

Edit [`src/data/profile.ts`](src/data/profile.ts) for career timeline, experience, and skills. Add a résumé PDF to `public/` and replace the hero email button with a direct link when ready.

## Before publishing

Please confirm the exact dates, company titles, awards, projects, education, and certifications, then add them to the data file. This initial release intentionally does not fabricate those details.
